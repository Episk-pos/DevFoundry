## Key Concepts

- Package managers: npm, yarn, pnpm
- package.json: scripts, dependencies, devDependencies
- TypeScript compiler (tsc) and ts-node
- Bundlers: Vite, webpack, esbuild
- Linters and formatters: ESLint, Prettier
