# Module 02: Anatomy of Projects

Understanding how software projects are organized.

**Full module**: [dev.episkopos.community/docs/curriculum/part-1-foundations/anatomy-of-projects](https://dev.episkopos.community/docs/curriculum/part-1-foundations/anatomy-of-projects)

## Key Concepts

- Language categories: compiled vs interpreted, typed vs untyped
- Architecture types: CLI, web app, API, library
- Project structure: source files, configuration, dependencies
- Tool chains: editors, runtimes, package managers
