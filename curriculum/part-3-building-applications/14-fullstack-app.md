# Stage 4: Fullstack App

Adding an Express backend with database persistence to the Lemonade Stand.

**Full module**: [episk-pos.github.io/DevFoundry/docs/curriculum/part-3-building-applications/fullstack-app](https://episk-pos.github.io/DevFoundry/docs/curriculum/part-3-building-applications/fullstack-app)

## Key Concepts

- Express.js API server
- REST API design
- SQLite database with better-sqlite3
- Connecting React frontend to backend API
- Environment variables
- Client-server architecture

## What You'll Build

A fullstack lemonade stand with:
- Express API server with REST endpoints
- SQLite database for orders
- React frontend calling the API
- Proper separation of client and server

## Exercises

4 exercises covering API endpoints, database queries, frontend integration, and error handling.
