# Stage 3: React SPA

Converting the Lemonade Stand to a modern React single-page application.

**Full module**: [devfoundry.io/docs/curriculum/part-3-building-applications/react-spa](https://devfoundry.io/docs/curriculum/part-3-building-applications/react-spa)

## Key Concepts

- Project setup with Vite
- Converting vanilla patterns to React components
- useState and useEffect hooks
- Component composition and props
- React Router for navigation
- Context for shared state

## What You'll Build

A React version of the lemonade stand with:
- Reusable components (MenuItem, OrderItem, etc.)
- Client-side routing (Menu, Checkout, Confirmation pages)
- Shared state with Context
- Modern tooling and hot reload

## Exercises

4 exercises covering component creation, state management, routing, and custom hooks.
