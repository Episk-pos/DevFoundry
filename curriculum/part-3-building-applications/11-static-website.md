# Stage 1: Static Website

Building the Lemonade Stand with HTML, CSS, and vanilla JavaScript.

**Full module**: [episk-pos.github.io/DevFoundry/docs/curriculum/part-3-building-applications/static-website](https://episk-pos.github.io/DevFoundry/docs/curriculum/part-3-building-applications/static-website)

## Key Concepts

- HTML structure and semantic elements
- CSS styling and layout (Flexbox, Grid)
- JavaScript DOM manipulation
- Event handling basics
- The browser as runtime environment

## What You'll Build

A functional lemonade stand that:
- Displays a menu of drinks
- Lets users add items to an order
- Shows order total
- Works entirely in the browser

## Exercises

4 exercises covering HTML structure, CSS styling, JavaScript interactivity, and debugging.
