# Stage 5: Deployed App

Deploying the Chat App to production — hosting, configuration, CI/CD, and real-time messaging.

**Full module**: [dev.episkopos.community/docs/curriculum/part-3-building-applications/deployed-app](https://dev.episkopos.community/docs/curriculum/part-3-building-applications/deployed-app)

## Key Concepts

- Production vs development environments
- Environment variables and configuration
- Deployment platforms (Railway, Render, Vercel)
- CI/CD with GitHub Actions
- WebSocket for real-time messaging
- Authentication with OAuth
- Monitoring and logging basics

## What You'll Build

A production-deployed chat app with:
- Frontend on Vercel/Netlify
- Backend on Railway/Render
- Real-time messaging with WebSockets
- OAuth authentication (GitHub/Google)
- Automated deployments from GitHub
